Puppy
=====

An opinionated cute little framework for bootstrapping Prism and MEF based WPF desktop apps.

Once complete, Puppy will possess these superpowers:

* Sparkle Updater
* Menu Items Service
* Package Manager Commands for zero-config bootstrapping
